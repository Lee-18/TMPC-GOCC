This is the code for developing a group thermal comfort model using a Random Forest (RF) model on a publicly available dataset.

**AllFullOccData_AtlantaTypSum.mat**: the dataset used for example  
**RFModel_TrainEvalue_TypSum.m**: the code used to develop the RF model and evaluate it using Jensen Shannon Divergence (JSD) as an index.  
**JensenShannonDivergence.m** and **MeanJSD.m**: the code using weighted averaging JSD for evaluation
